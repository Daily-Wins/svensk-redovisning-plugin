# K1 Ideella föreningar och registrerade trossamfund

Kunskapsbas för BFNAR 2010:1, uppdaterad 2024-12-13. Förenklat regelverk för ideella föreningar och registrerade trossamfund med nettoomsättning högst 3 miljoner kronor.

## När ska denna skill användas?

- Frågor om K1 för ideella föreningar / trossamfund
- Frågor om förenklat årsbokslut för ideell förening
- Frågor om medlemsavgifter, gåvor, bidrag i ideella föreningar
- Frågor om kontantmetoden för ideella föreningar
- Frågor om vad som ska bokföras i föreningen

## Vanliga frågor

| Fråga | Svar |
|---|---|
| Vem får använda K1 ideella? | Ideella föreningar/trossamfund med nettoomsättning ≤3 mkr (punkt 1.1) |
| Hur redovisas medlemsavgifter? | Som intäkt när de betalas in (kontantmetoden) eller faktureras (punkt 7.3) |
| Hur redovisas erhållna gåvor? | Gåvor av materiell karaktär tas upp till marknadsvärde (punkt 3.4) |
| Hur redovisas bidrag? | Villkorade bidrag: intäkt vid förbrukning. Ovillkorade: vid mottagande (punkt 7.5-7.7) |
| Måste vi periodisera? | Nej, kontantmetoden tillåts — bokföring vid betalning |
