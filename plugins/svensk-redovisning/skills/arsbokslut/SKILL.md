# BFNAR 2017:3 Årsbokslut

## Viktigt

Detta är **Årsbokslut** (BFNAR 2017:3), INTE **Årsredovisning** (BFNAR 2016:10).

- **Årsbokslut** = resultaträkning + balansräkning (BFL 6 kap.)
- **Årsredovisning** = förvaltningsberättelse + resultaträkning + balansräkning + noter (ÅRL)

Företag som upprättar årsredovisning (aktiebolag, ekonomiska föreningar m.fl.) ska använda K2 Årsredovisning (skill: k2-regelverk) eller K3 (skill: k3-regelverk).

## Instruktioner

1. Läs relevant kapitel-fil baserat på frågeområdet i tabellen ovan.
2. Citera exakt punktnummer (t.ex. "punkt 5.16") vid hänvisning.
3. Ange om regeln är Lagtext, Allmänt råd eller Kommentar.
4. Kontrollera alltid om det finns särskilda regler per företagsform (enskild firma, handelsbolag, ideell förening, samfällighetsförening, stiftelse, filial).
5. Punkt 1.7 möjliggör tillämpning av K3 (BFNAR 2012:1) – hänvisa till skill k3-regelverk vid behov.
6. Regelverket ska tillämpas i sin helhet (punkt 1.5) – cherry-picking är inte tillåtet.
