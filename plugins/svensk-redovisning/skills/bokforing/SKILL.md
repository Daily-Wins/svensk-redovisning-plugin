# Bokföring

Kunskapsbas för BFNAR 2013:2, uppdaterad 2024-09-16. Grundläggande regelverk för löpande bokföring som gäller alla bokföringsskyldiga företag oavsett K-regelverk.

## När ska denna skill användas?

- Frågor om löpande bokföring, bokföringsordning
- Frågor om verifikationer (kvitton, fakturor, innehåll, numrering)
- Frågor om arkivering av räkenskapsinformation (hur, var, hur länge)
- Frågor om anläggningsregister
- Frågor om gemensam verifikation, kontantförsäljning
- Frågor om systemdokumentation och behandlingshistorik
- Frågor om tidpunkt för bokföring (kontantmetod vs faktureringsmetod)
- Frågor om räkenskapsinformation i elektronisk form

## Vanliga frågor

| Fråga | Svar |
|---|---|
| Hur snabbt måste jag bokföra? | Kontanta: senast påföljande arbetsdag. Övriga: "så snart det kan ske" (punkt 3.1-3.5) |
| Får jag använda kontantmetoden? | Ja om nettoomsättning ≤3 mkr och ≤50 verifikationer/månad (punkt 3.12) |
| Vad ska en verifikation innehålla? | Datum, belopp, motpart, vad det avser, kontering (punkt 5.6-5.7) |
| Hur länge ska jag spara bokföringen? | 7 år efter räkenskapsårets utgång (punkt 8.9) |
| Var ska räkenskapsinformationen förvaras? | I Sverige, i ordnat skick, tillgänglig (punkt 8.4-8.5) |
| Vad är systemdokumentation? | Beskrivning av bokföringssystemet, kontoplan, verifikationsserier (punkt 9.1-9.5) |
| Måste jag ha anläggningsregister? | Ja om du har materiella anläggningstillgångar (punkt 4.3) |
