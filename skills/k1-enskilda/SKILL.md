# K1 Enskilda näringsidkare

Kunskapsbas för BFNAR 2006:1, uppdaterad 2025-03-24. Förenklat regelverk för enskilda näringsidkare med nettoomsättning högst 3 miljoner kronor.

- Frågor om NE-bilagan / deklaration för enskild firma

## Vanliga frågor

| Fråga | Svar |
|---|---|
| Vem får använda K1? | Enskild näringsidkare med nettoomsättning ≤3 mkr (punkt 1.1) |
| Kontantmetoden eller faktureringsmetoden? | Valfritt, men konsekvent. Kontantmetoden: bokför vid betalning (punkt 4.2-4.5) |
| Hur värderas inventarier? | Anskaffningsvärde minus avskrivning. Räkenskapsenlig eller restvärdesavskrivning (punkt 6.35-6.38) |
| Måste jag periodisera? | Nej, K1 tillåter kontantmetoden — ingen periodisering krävs |
| Hur hanteras privata uttag? | Uttag = minskning av eget kapital, inte kostnad (punkt 3.12-3.15) |
